Try this sequence before deeper debugging:

1. Confirm iOS is updated.
2. Re-open the app and keyboard.
3. Test exact emoji names and then short phrase variants.
4. If search still misses phrase intent, switch to a meaning-first search flow.

Native keyboard search can miss social phrasing. That is where FetchMoji tends
to be faster for “emoji for X” style queries.
