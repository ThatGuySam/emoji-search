## SBC4

- `Tease:` One privacy boundary connects search, content, extension reuse, and measurement.
- `Lede:` The user searches locally, learns from reviewed pages, reuses the extension, and contributes only privacy-safe lifecycle signals.
- `Why it matters:` The end-to-end flow prevents a growth optimization from silently weakening the product promise.
- `Go deeper:` Follow the five stages from search through compounding discovery.

## 1. Search by meaning

The search field accepts a natural phrase. It preserves the text immediately,
shows an honest initialization state when the local index is not ready, and
ranks a compact result set without transmitting the query.

Each result shows the emoji, a short tone label, and a copy action. Copy success
is visible and announced to assistive technology.

## 2. Learn from an intent page

Search engines and answer engines land on server-rendered phrase pages. Each
page gives one fast default, compares nearby tones, provides copy-ready message
examples, and explains when a candidate can be misread.

Pages are created from observed demand and reviewed for actual distinct value.
Localized pages do not publish until a native reviewer approves the nuance.

## 3. Reuse it where writing happens

The browser extension opens the same search from a toolbar button or keyboard
shortcut. It does not read page content. Search stays local, and one action
copies the choice back to the clipboard.

## 4. Measure the funnel without reading the message

The product can measure page arrival, search readiness, result display, copy,
and return use. Events never contain the query, surrounding page text, or the
message being written. Public privacy copy states the actual boundary rather
than making an absolute telemetry claim the deployment cannot support.

## 5. Compound discovery

Search Console evidence selects which pages to improve. The extension, technical
case study, short demonstrations, and relevant communities create distribution
that can also earn real mentions and links. Product Hunt is a supporting launch
event after the stronger product surface exists.
