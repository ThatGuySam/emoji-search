## SBC4

- `Tease:` Review the next FetchMoji growth iteration before implementation.
- `Lede:` The site joins product behavior, search discovery, privacy, extension distribution, and their source research in one spec.
- `Why it matters:` The live product already works, but the planned changes cross boundaries that can easily drift when handled separately.
- `Go deeper:` Start with the Introduction, inspect the Roadmap, and use the Research section to audit the evidence.

:::note[The product is live; this growth iteration is planned]
[FetchMoji](https://fetchmoji.com/) already provides private, client-side emoji
search. These docs describe the reviewed target state for the next growth
iteration as if it works. Features marked _Planned_ are not claims about the
current site. The machine-readable spec lives at
[`/llms-full.txt`](/llms-full.txt).
:::
